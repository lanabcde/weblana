Percabangan If..Else

Percabangan if..else adalah salah satu bentuk percabangan dalam pemrograman yang digunakan untuk memeriksa apakah sebuah kondisi bernilai True atau False. Jika kondisi bernilai True, maka blok kode pada bagian if akan dieksekusi, dan jika kondisi bernilai False, maka blok pernyataan pada bagian else akan dieksekusi. Perhatikan flowchart dibawah ini untuk memahami bagaimana percabanagan if…else bekera :

Flowchart diatas menjelaskan jika kondisi bernilai benar maka program akan memproses blok kode if, namun jika kondisi bernilai salah maka blok kode pada bagian else yang akan diproses. Berikut struktur penulisan percabangan if..else :

Struktur Dasar Percabangan If..Else

If kondisi :
  Perintah_jika_kondisi_benar
  ……………………………………………..
Else :
  Perintah_jika_kondisi_salah
 ……………………………………………..

Contoh Program menggunakan percabangan if..else :

Contoh 2.2 Percabangan If..Else

nilai = 50
if nilai <=70 :
  print("Maaf Anda Gagal")
else :
  print("Selamat Anda Lulus")
