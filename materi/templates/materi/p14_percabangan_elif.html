Percabangan Elif

Percabangan elif dalam Python adalah salah satu bentuk percabangan yang digunakan ketika kita memiliki beberapa kondisi yang ingin diperiksa. Penggunaan elif memungkinkan kita untuk memeriksa kondisi tambahan jika kondisi sebelumnya bernilai false. Dengan kata lain, jika kondisi pada blok if bernilai false, maka kondisi pada blok elif akan dievaluasi. Jika kondisi pada blok elif juga bernilai false, maka blok else akan dieksekusi. Perhatikan flowchart dibawah ini untuk memahami bagaimana percabanagan elif bekerja :

Flowchart diatas menjelaskan bagaimana program akan memeriksa dari kondisi pertama yaitu kondisi if, jika kondisi bernilai benar maka blok kode if yang akan diproses, jika kondisi bernilai salah maka blok kode elif yang akan diproses terus menerus sasmpai kondisi elif bernilai salah, jika kondisi pada elif bernilai salah maka blok kode else yang akan diproses berikut struktur penulisan percabangan elif :

Struktur Dasar Percabangan Elif

if kondisi :
  Perintah_jika_kondisi_benar
elif (kondisi2):
  perintah_jika_kondisi2_benar
elif (kondisi3):
  perintah_jika_kondisi3_benar
..................................
else :
  perintah_jika_semua_kondisi_Salah
