Tipe Data

Dalam bahas pemrograman tipe data adalah jenis-jenis data yang akan di olah oleh komputer. Dalam bahasa python ada dua jenis tipe data yaitu tipe data primitive dan tipe data user defined. Tipe data primitive adaah tipe data yang sudah disediakan secara default oleh Python. Tipe data primitif yang ada di python adalah str, int, float, bool, dll.

  • String

    • String adalah tipe data yang hanya digunakan untuk memproses text. Di dalam Python, String merupakan tipe data yang digunakan untuk merepresentasikan teks atau urutan karakter. String dapat berisi satu atau lebih karakter, dan dapat terdiri dari huruf, angka, tanda baca, dan simbol lainnya. String dapat dibuat dengan dua cara: menggunakan tanda kutip tunggal (' ') atau tanda kutip ganda (" "). Contoh penggunaan tipe data string :

    'lana'
    "Muhammad Maulana"

  • Integer

    • Tipe data integer adalah tipe data yang digunakan untuk memproses bilangan bulat. Tipe data integer dapat menampung bilangan bulat positif, nol, bilanga bulat negative. Dalam Python, integer digunakan untuk merepresentasikan bilangan bulat dan dapat melakukan operasi matematika seperti penjumlahan, pengurangan, perkalian, pembagian, modulus, dan pangkat. Kita juga dapat mengkonversi integer ke tipe data lain, membulatkan bilangan, mengubah basis bilangan, menghitung panjang bilangan, dan membuat range bilangan untuk iterasi dalam loop. Berikut contoh tipe data integer :

    7
    10
    200
    4000

  • Float

    • Float adalah salah satu tipe data numerik dalam bahasa pemrograman Python yang digunakan untuk merepresentasikan bilangan real dengan titik desimal. Tipe data float dapat digunakan untuk melakukan operasi matematika seperti penjumlahan, pengurangan, perkalian, dan pembagian. Berikut contoh penggunaan tipe data string :

    3.14
    20.211
    172.3

  • Boolean

    • Tipe data Boolean dalam Python adalah tipe data yang hanya memiliki dua nilai: True dan False. Nilai True dan False digunakan untuk mengevaluasi kondisi atau ekspresi logika dalam program Python. Dalam bahasa pemrograman lain, tipe data ini sering disebut dengan tipe data boolean atau tipe data logika. Berikut ontoh penggunaan tipe data Boolean dalam Python adalah untuk menguji apakah suatu kondisi benar atau salah :

    True
    False
