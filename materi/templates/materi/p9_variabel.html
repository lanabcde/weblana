Variabel

Dalam matematika variable adalah “wadah “ untuk menyimpan suatu nilai. Dalam pemrograman variable memiliki kegunaan yang sama yaitu untuk menyimpan suatu nilai agar bisa diolah oleh komputer. Dalam bahasa python memiliki aturan-aturan untuk mendefinisikan variable. Berikut aturan-aturan mendefinisiakn variable :

  • Nama variabel harus dimulai dengan huruf atau garis bawah (_).
  • Nama variabel tidak boleh dimulai dengan angka.
  • Nama variabel hanya dapat mengandung karakter alfanumerik dan garis bawah.
  • Nama variabel bersifat case sensitive, sehingga variabel "nama" dan "Nama" dianggap berbeda.
  • Hindari menggunakan kata kunci (keyword) Python seperti "if", "else", "while",dan sebagainya sebagai nama variabel.
  • Usahakan memberikan nama variabel yang deskriptif, sehingga mudah dipahami dan dibaca.

Berikut adalah Contoh penulisan variabel yang benar :

umur = 20
nama_lengkap = "Muhammad Maulana"
jumlahsiswa = 34

Berikut adalah contoh penulisan variabel yang salah:

1umur = 27 # tidak boleh dimulai dengan angka
nama lengkap = "John Doe" # Tidak boleh menggunakan Spasi
JumlahSiswa = 34 # tidak konsisten dalam penggunaan huruf kapital
if = 50 # menggunakan kata kunci Python sebagai nama variabel
