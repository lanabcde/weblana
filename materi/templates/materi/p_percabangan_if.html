PERCABANGAN IF

Percabangan if dalam Python adalah salah satu konsep dasar dalam pemrograman yang digunakan untuk mengeksekusi sebuah blok kode jika kondisi tertentu bernilai True. Blok kode tersebut tidak akan dieksekusi jika kondisi tersebut bernilai False. Perhatihan flowcahr dibawah ini untuk lebih memahami bagaimana percabangan if bekerja


Flowchart diatas menjelakan bagaimana percabanagn if bekerja, dimulai dari membaca inputan yang diberikan lalau diperiksa apakah inputan tersebut memenuhi kondisi yang ditentukan, jika kondisi yang ditentukan terpenuhi maka pada blok kode akan diperoses. Namun jika kondisi tidak terpenuhi maka program akan langsung berhenti tanpa memproses blok kode if. Berikut struktur penulisan if :
